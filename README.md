# React ToDo List
### An easy-to-use SPA with the following features: 
- create new task
- edit task
- check if the task has been done
- remove task
- clear form
- remove all tasks

#### Stack:
- React
- React Router
- React Bootstrap
- hoc
- localStorage
- HTML
- Sass
- Formik, Yup, ESLint
- class Storage created to emulate work through the API
